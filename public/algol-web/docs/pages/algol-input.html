<algol-input>

Visão Geral

Componente fundamental de entrada de dados (texto). Fornece estilo consistente com estado de foco ("glow"), bloqueio interativo quando desativado e integração robusta com validações de formulário.

<algol-input type="email" label="E-mail" placeholder="usuario@email.com" required></algol-input>

Atributos de Configuração

Atributo Descrição Valores Aceitos
label Texto descritivo exibido acima do campo de entrada. Texto
type Define a natureza dos dados e o tipo de teclado (mobile). Se um tipo inválido for passado, o componente reverte para texto simples. text (padrão), email, password, url, search
value Valor de texto atual do input. Texto
placeholder Texto em tom menor exibido enquanto o campo está vazio. Texto
minlength Quantidade mínima de caracteres exigida. Número inteiro
maxlength Limite máximo de caracteres permitidos. Número inteiro
pattern Expressão Regular (Regex) que o texto deve respeitar para ser considerado válido. Expressão Regular (Texto)
disabled Bloqueia interações, altera o cursor e escurece o campo visualmente. Booleano (presença do atributo)
required Invalida o componente se submetido em branco. Booleano (presença do atributo)

Nota: Este componente utiliza as variáveis globais de tema e não possui atributos de estilo em linha mapeados.

Eventos Emitidos

Além dos eventos herdados do BaseComponent, o input emite eventos específicos para capturar a digitação do usuário:

  • algol-input: Disparado em tempo real a cada tecla pressionada. Ideal para máscaras, buscas ao vivo ou contadores de caracteres. 
    document.querySelector('algol-input').addEventListener('algol-input', e => {
  console.log("Digitando agora:", e.detail.value);
});
  • algol-change: Disparado apenas quando o campo perde o foco (blur) e teve seu conteúdo alterado. 
    document.querySelector('algol-input').addEventListener('algol-change', e => {
  const { oldValue, value } = e.detail;
  console.log(`Foco perdido. Mudou de [${oldValue}] para [${value}]`);
});

Validação Interna e ElementInternals

Integração com <form>: O algol-input é um cidadão nativo de formulários (formAssociated: true). Ele traduz as restrições inseridas (required, minlength, pattern, type="email") para estados de erro internos (valueMissing, typeMismatch, tooShort, patternMismatch), bloqueando a submissão do formulário caso o texto esteja inválido.
