BaseCheckable

Visão Geral

A classe BaseCheckable é uma extensão do BaseComponent. Ela não é instanciada diretamente, servindo como a "espinha dorsal" lógica para componentes de seleção de estado duplo: algol-checkbox, algol-radio e suas respectivas versões switch.

Sua principal função é gerenciar a sincronização do estado checked, o vínculo com a API de validação nativa de formulários e emitir eventos de alteração padronizados para seus herdeiros.

Atributos Padrão Herdados

Qualquer componente que derive do BaseCheckable suportará nativamente os seguintes atributos:

Atributo Descrição Valores Aceitos
label Texto de clique exibido ao lado da caixa/botão de seleção. Texto
checked Define se o componente inicia marcado. Booleano (presença do atributo)
name Nome do campo usado no agrupamento (ex: Radios) e no envio de formulários. Texto
value O valor que será enviado pelo formulário caso o componente esteja marcado. Texto (Se omitido, envia "on" por padrão)
disabled Desabilita a interação de clique e altera o cursor para bloqueado. Booleano (presença do atributo)
required Exige que o componente seja marcado para que o formulário possa ser enviado (Útil para "Aceito os termos"). Booleano (presença do atributo)

Motor de Formulário (ElementInternals)

Gestão de Estado: A classe esconde um <input> nativo no Shadow DOM. Quando o usuário interage visualmente com o componente, o BaseCheckable altera o input escondido, recolhe seu estado e avisa ao formulário (via setFormValue) se deve enviar o value ou ignorá-lo (null). Se for required e não estiver marcado, ele emite o erro valueMissing internamente.

Eventos Emitidos

Além dos eventos básicos (algol-click, algol-focus) injetados pelo BaseComponent, o BaseCheckable introduz o evento de troca de estado:

  • algol-input: Disparado sempre que o estado checked muda via interação do usuário. Retorna o valor caso esteja marcado, ou null caso tenha sido desmarcado. 
    componente.addEventListener('algol-input', e => {
  const { origin, value } = e.detail;
  const isChecked = origin.checked; // Acessando a propriedade booleana
  
  console.log(`Estado: ${isChecked ? 'Marcado' : 'Desmarcado'} | Valor: ${value}`);
});
