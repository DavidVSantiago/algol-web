<algol-input-range>

Visão Geral

Componente deslizante (slider) para seleção de valores numéricos contínuos ou em etapas. Ele fornece uma interface amigável e unificada, eliminando as inconsistências visuais drásticas que existem entre diferentes navegadores.

<algol-input-range label="Volume" min="0" max="100" step="10" value="50"></algol-input-range>

Atributos de Configuração

Herdando da classe base de Input, ele compartilha atributos como label e disabled, mas seu comportamento central é governado pelas propriedades matemáticas nativas do slider:

Atributo Descrição Valores Aceitos
value O valor numérico atual da posição da bolinha (thumb). Número (Padrão nativo costuma ser o valor médio entre o min e o max)
min O valor mínimo (extrema esquerda do trilho). Número (Padrão: 0)
max O valor máximo (extrema direita do trilho). Número (Padrão: 100)
step Granularidade do controle. Define de quanto em quanto o valor avança ou retrocede. Número (Padrão: 1)
disabled Trava a bolinha e altera a cor do trilho para indicar inatividade. Booleano (presença do atributo)

Estilização Cross-Browser Unificada

Engenharia Visual: Um dos maiores desafios do CSS web é o input range. A Algol UI contorna isso implementando estilizações diretas no Shadow DOM para as shadow parts nativas de diferentes motores: ::-webkit-slider-runnable-track e ::-webkit-slider-thumb (Chrome/Safari/Edge), e ::-moz-range-track e ::-moz-range-thumb (Firefox), garantindo que o componente pareça e se comporte exatamente igual em qualquer plataforma.

Eventos Emitidos

Este componente tem uma dinâmica de eventos dividida entre interação em tempo real (arrastar) e a confirmação (soltar):

  • algol-input: Disparado continuamente de forma ininterrupta enquanto o usuário mantém o clique pressionado e arrasta a bolinha. Excelente para atualizar visualizações em tempo real (como mostrar o número do volume mudando dinamicamente na tela). 
    document.querySelector('algol-input-range').addEventListener('algol-input', e => {
  console.log("Arrastando. Valor atual:", e.detail.value);
});
  • algol-change: Disparado exclusivamente quando o usuário encerra a interação (solta o clique do mouse ou levanta o dedo da tela em dispositivos touch), sinalizando que o valor final foi decidido. 
    document.querySelector('algol-input-range').addEventListener('algol-change', e => {
  const { oldValue, value } = e.detail;
  console.log(`Seleção concluída. Movido de [${oldValue}] para [${value}]`);
});

Integração com Formulários

Utiliza a validação interna do navegador para interceptar falhas lógicas, comunicando o estado stepMismatch (se o valor inserido via script não respeitar o intervalo) ou limites ultrapassados (rangeUnderflow e rangeOverflow).
