<algol-grid-item>

Visão Geral

O algol-grid-item é uma célula estrutural projetada para ser utilizada dentro do <algol-grid-layout>. Ele permite controle rápido de expansão (span), alinhamento isolado da célula, espaçamento interno responsivo e suporta um sistema avançado de imagem de fundo com overlay embutido.

Nota interna: O próprio componente renderiza como display: grid com gap: 0 por padrão, permitindo que seu conteúdo interno também seja alinhado facilmente.

<algol-grid-item colspan="2" posh="center" padding="2" color="#f0f0f0">
    Conteúdo da Célula
</algol-grid-item>

Dimensão, Alinhamento e Espaçamento

Estes atributos controlam a geometria do item na grade e como seu conteúdo respira:

Atributo Descrição / Regra Injetada
colspan Expansão horizontal (grid-column-end: span [valor]).
rowspan Expansão vertical (grid-row-end: span [valor]).
posh Alinhamento horizontal próprio (justify-self). Ex: center, end, start.
posv Alinhamento vertical próprio (align-self). Ex: center, end, start.
padding Espaçamento interno. Converte números soltos em vw automaticamente. (Ex: "1 2" vira 1vw 2vw).
color Define a cor de fundo (background-color) da célula.

Controle Responsivo (Mobile)

Sufixo break: Todos os atributos listados acima possuem uma versão terminada em break (ex: colspanbreak, poshbreak, paddingbreak, colorbreak). Eles ativam Media Queries que sobrescrevem os valores nativos exclusivamente quando a tela atinge a largura mobile.

Sistema Avançado de Background (Imagem)

Overlay Inteligente: Ao utilizar o atributo imgoverlay passando um valor entre 0 e 1, o componente gera automaticamente um gradiente escuro por cima da imagem de fundo para garantir contraste do texto, sem necessidade de divs extras.
Atributo Função (Mapeamento CSS)
img URL da imagem de fundo (background-image: url(...)).
imgoverlay Nível de escurecimento sobre a imagem (ex: 0.5 para 50% preto).
imgsize Tamanho do fundo (ex: cover, contain).
imgpos Posição da imagem (ex: center, top right).
imgrepeat Repetição (ex: no-repeat, repeat-x).
imgattach Fixação do fundo (ex: fixed para efeito parallax parallax simples).
