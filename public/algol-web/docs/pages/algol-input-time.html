<algol-input-time>

Visão Geral

Componente especializado para seleção de horários. Aciona o relógio/seletor nativo do sistema operacional e possui suporte a limitação de horas (mínimo/máximo) e controle de intervalos de seleção.

<algol-input-time label="Horário da Reunião" min="08:00" max="18:00" step="900"></algol-input-time>

Atributos de Configuração

Herda as propriedades base (label, disabled, required, value) e adiciona os seguintes controles matemáticos e de tempo:

Atributo Descrição Valores Aceitos
min O horário mais cedo permitido. Texto no formato HH:MM (ex: 08:30)
max O horário limite permitido. Texto no formato HH:MM (ex: 18:00)
step Define o intervalo de precisão em segundos. Omitir este atributo limita a seleção apenas a horas e minutos. Se usar step="1", o seletor exibirá a casa dos segundos. Para saltos de 15 minutos, use 900. Número inteiro (Segundos)

Mecânicas Específicas

Filtro de Evento: Assim como o seletor de datas, o algol-input-time desativa a emissão do evento em tempo real (_canEmitInput = false). Isso evita disparos em massa na árvore do DOM enquanto o usuário está girando as roletas ou os ponteiros do relógio nativo do sistema. O valor só é despachado quando a escolha é concluída.

Eventos Emitidos

O monitoramento deve ser feito capturando o evento de mudança consolidada:

  • algol-change: Disparado apenas quando o usuário finaliza a escolha do horário e o componente perde o foco ou a janela do relógio é fechada. 
    document.querySelector('algol-input-time').addEventListener('algol-change', e => {
  const { oldValue, value } = e.detail;
  // O value segue o formato de 24h (ex: "14:30" ou "14:30:15")
  console.log(`Horário confirmado: ${value}`); 
});

Validação Interna

O algol-input-time é estritamente ligado à API ElementInternals do formulário. Ele bloqueará a submissão nativamente reportando:

  • rangeUnderflow: Se o horário escolhido for menor que o min.
  • rangeOverflow: Se for maior que o max.
  • stepMismatch: Se o horário inserido não for um múltiplo válido do step configurado.
