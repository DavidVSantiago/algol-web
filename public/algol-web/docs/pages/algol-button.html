<algol-button>

Visão Geral

O componente de botão padrão da biblioteca Algol. Suporta estados de carregamento, desativação nativa e integração automática com formulários.

<algol-button size="mid">Confirmar</algol-button>

Atributos de Configuração

Atributo Descrição Valores Aceitos
size Define variações de tamanho do botão. small, mid (padrão), big
type Comportamento do botão dentro de um formulário. button (padrão), submit, reset
disabled Desabilita interações visualmente e bloqueia eventos. Booleano (presença do atributo)
loading Oculta o texto, exibe um spinner e bloqueia cliques. Booleano (presença do atributo)
name Nome de identificação do input no formulário. Texto
value Valor associado ao botão em submissões. Texto

Atributos de Estilização

Atributo Variável Injetada Efeito Visual
color --button-color Substitui a cor de fundo padrão (--accent-A).
textcolor --button-textcolor Substitui a cor da fonte padrão (--text-C).

Eventos Emitidos

Este componente herda automaticamente todos os eventos do BaseComponent (algol-focus, algol-mouseenter, etc.). O principal evento utilizado na prática é o de clique:

  • algol-click: Disparado ao clicar (ignorado se disabled ou loading). 
    document.getElementById('btn').addEventListener('algol-click', e => {
  const { origin, mouse: { x, y } } = e.detail;
  console.log("Botão clicado:", origin, "Coords:", x, y);
});

Integração com Formulários

formAssociated: true
Este componente é nativamente reconhecido por tags <form>. Se você definir type="submit", ele enviará o formulário pai automaticamente. Se definir type="reset", limpará todos os campos.
