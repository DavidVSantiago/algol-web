<algol-textarea>

Visão Geral

Componente de área de texto expansível. Suporta controle de redimensionamento, associação automática com formulários e validação nativa de caracteres mínimos/máximos.

<algol-textarea label="Mensagem" lines="4" placeholder="Digite aqui..."></algol-textarea>

Atributos de Configuração

Atributo Descrição Valores Aceitos
label Texto descritivo exibido acima do campo de texto. Texto
value Define ou recupera o conteúdo de texto atual. Texto
placeholder Texto de dica exibido quando o campo está vazio. Texto
lines Define a altura inicial baseada no número de linhas (injeta o atributo rows). Número inteiro
fixed Trava o redimensionamento manual. Remove a alça puxável do canto (resize: none). Booleano (presença do atributo)
maxlength Limite máximo de caracteres permitidos na digitação. Número inteiro
readonly Torna o campo somente leitura. O usuário pode focar e copiar, mas não editar. Booleano (presença do atributo)
disabled Desabilita o campo completamente, alterando o visual e bloqueando eventos. Booleano (presença do atributo)
required Marca o campo como obrigatório para a submissão de formulários. Booleano (presença do atributo)

Comportamento de Preenchimento (Slot)

Preenchimento Direto: Você pode injetar o valor inicial escrevendo diretamente dentro da tag. O componente escuta o evento slotchange e atribui o conteúdo ao value.
Exemplo: <algol-textarea>Texto inicial injetado direto no HTML</algol-textarea>

Eventos Emitidos

Além de herdar os eventos do BaseComponent (foco, mouse, etc.), este componente dispara eventos cruciais de edição de texto:

  • algol-input: Disparado em tempo real a cada tecla digitada ou alteração no texto. 
    document.querySelector('algol-textarea').addEventListener('algol-input', e => {
  console.log("Texto atualizado em tempo real:", e.detail.value);
});
  • algol-change: Disparado apenas quando o campo perde o foco e o valor foi alterado. Útil para salvar dados sem sobrecarregar o sistema. 
    document.querySelector('algol-textarea').addEventListener('algol-change', e => {
  const { oldValue, value } = e.detail;
  console.log(`Alterado de [${oldValue}] para [${value}]`);
});

Integração e Validação

Possui formAssociated: true. Se o atributo required estiver presente e o campo estiver vazio, ou se o maxlength for ultrapassado (via injeção de script), ele comunica o estado de erro (valueMissing ou tooLong) diretamente ao formulário pai através da API ElementInternals.
