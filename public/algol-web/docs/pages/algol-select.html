<algol-select>

Visão Geral

Componente de seleção suspensa (dropdown) customizado. Oferece estilo uniforme, seta personalizada e integração completa com a API de validação de formulários do navegador.

<algol-select label="Escolha um item" placeholder="Selecione...">
    <option value="1">Opção A</option>
    <option value="2">Opção B</option>
</algol-select>

Atributos de Configuração

Atributo Descrição Valores Aceitos
label Texto descritivo exibido acima do campo. Texto
value Define ou recupera o valor da opção atualmente selecionada. Texto correspondente a um value de uma <option>
placeholder Texto exibido como opção padrão desabilitada (ex: "Select..."). Texto
disabled Desabilita a interação com o menu suspenso. Booleano (presença do atributo)
required Torna a seleção obrigatória. A validação falha se a opção placeholder estiver selecionada. Booleano (presença do atributo)

Nota: Este componente utiliza as variáveis globais de tema da Algol UI e não possui atributos de estilização independentes (como color).

Injeção de Opções (Slot)

Sincronização Inteligente: Para popular o select, basta inserir tags <option> padrão dentro do componente. A Algol detecta essas opções via slotchange, clona-as para o select interno de forma segura e injeta automaticamente uma primeira opção invisível/desabilitada baseada no seu atributo placeholder.

Eventos Emitidos

Assim como os demais componentes, ele herda o mapeamento do BaseComponent (focus, blur, etc.). Para capturar trocas de valor, utilize o evento algol-change:

  • algol-change: Disparado no exato momento em que o usuário seleciona uma nova opção no menu suspenso (ignorado se o componente estiver disabled). 
    document.querySelector('algol-select').addEventListener('algol-change', e => {
  const { origin, oldValue, value } = e.detail;
  console.log(`Valor alterado de [${oldValue}] para [${value}] no componente:`, origin);
});

Validação de Formulário

O componente suporta formAssociated: true. Se marcado como required, o formulário identificará uma falha de validação (valueMissing) caso o usuário tente submeter sem ter alterado o valor padrão do placeholder.
