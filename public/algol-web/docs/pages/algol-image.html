<algol-image>

Visão Geral

Componente de imagem inteligente. Aplica object-fit: cover por padrão, gerencia falhas de carregamento automaticamente (exibindo um aviso visual) e suporta controle de estilização diferenciado para modo mobile.

<algol-image src="foto.jpg" alt="Descrição da foto" lazy radius="0.5vw"></algol-image>

Atributos de Configuração

Atributo Descrição Valores Aceitos
src URL ou caminho da imagem. Texto (URL)
alt Texto alternativo de acessibilidade. Texto
width Define o atributo nativo de largura da imagem. Número/Texto
height Define o atributo nativo de altura da imagem. Número/Texto
lazy Ativa o carregamento sob demanda (loading="lazy"). Sem ele, o padrão é eager. Booleano (presença do atributo)

Atributos de Estilização

Atributo Variável Injetada Efeito Visual
radius --image-radius Define o arredondamento dos cantos (border-radius).
radiusbreak --image-radiusbreak Define o arredondamento exclusivo para telas menores que o breakpoint mobile.
size --image-size Define a largura CSS da imagem.
sizebreak --image-sizebreak Define a largura CSS exclusiva para telas menores que o breakpoint mobile.

Comportamento e Fallback

Se o atributo src estiver quebrado ou a imagem não for encontrada, o componente oculta a tag <img> e exibe automaticamente a mensagem visual: ⚠️ Image not loaded!, disparando também um erro de validação (badInput).

Eventos Emitidos

Além dos eventos herdados do BaseComponent (como algol-click), a imagem emite eventos específicos de seu ciclo de vida:

  • algol-image-load: Disparado assim que a imagem é carregada com sucesso. 
    document.querySelector('algol-image').addEventListener('algol-image-load', e => {
  console.log("Imagem carregada com sucesso:", e.target);
});
  • algol-image-error: Disparado se houver falha no carregamento do arquivo. 
    document.querySelector('algol-image').addEventListener('algol-image-error', e => {
  console.error("Falha ao carregar a imagem do componente:", e.target);
});
