<algol-switch-checkbox>

Visão Geral

Componente de interruptor (Toggle Switch). Visualmente, ele exibe uma interface moderna de deslizar (on/off), ideal para painéis de configurações. Mecanicamente, ele é idêntico a um checkbox padrão, herdando toda a robustez da classe BaseCheckable.

<algol-switch-checkbox name="notificacoes" checked>Ativar notificações</algol-switch-checkbox>

Diferencial Semântico

Quando usar? Enquanto o <algol-checkbox> é ideal para múltiplos itens em uma lista ou aceite de termos, o <algol-switch-checkbox> passa a sensação de "mudança instantânea de estado". É a escolha perfeita para ativar Dark Mode, ligar/desligar Wi-Fi em um painel ou habilitar configurações de usuário.

Atributos de Configuração

Por ser um herdeiro direto do BaseCheckable, ele compartilha exatamente os mesmos atributos do checkbox convencional:

  • checked: Faz a bolinha iniciar na posição "Ligado" (à direita, colorida).
  • name: Nome do campo no formulário.
  • value: O valor enviado quando está na posição "Ligado" (padrão "on").
  • disabled: Trava a animação, escurece o componente e impede o clique.
  • required: Impede o envio do formulário se o interruptor estiver desligado.
  • label: Texto embutido (embora o uso de <slot> seja recomendado para textos clicáveis).

Eventos Emitidos

A captura de dados é feita da mesma forma, escutando a mudança de estado da chave:

  • algol-input: Disparado no momento em que a chave termina a animação de deslizar. 
    document.querySelector('algol-switch-checkbox').addEventListener('algol-input', e => {
  const isLigado = e.detail.origin.checked;
  console.log(`Interruptor alterado. Estado atual: ${isLigado ? 'LIGADO' : 'DESLIGADO'}`);
});
