<algol-input-number>

Visão Geral

Componente especializado para entrada de dados numéricos. Ele estende o <algol-input> padrão, adicionando controles giratórios (spinners) laterais, suporte avançado a teclado e filtros rigorosos contra caracteres não numéricos.

<algol-input-number label="Idade" min="0" max="120" value="18"></algol-input-number>

Atributos de Configuração

Além de herdar os atributos padrão de input (label, placeholder, disabled, required), este componente adiciona propriedades matemáticas:

Atributo Descrição Valores Aceitos
min Define o valor mínimo permitido. Número
max Define o valor máximo permitido. Número
value Valor numérico atual. Aceita decimais. Número

Mecânicas de Validação e UX

Filtro em Tempo Real: O componente bloqueia nativamente a digitação de letras. Apenas números, pontos e vírgulas são aceitos. Ao perder o foco, vírgulas são convertidas para pontos automaticamente (padrão flutuante).
Trava de Limites (Min/Max): Se o usuário digitar um valor acima do max ou abaixo do min, o componente fará a correção automática para o limite mais próximo assim que o campo perder o foco (blur) ou as setas de controle forem clicadas.
  • Controles UI: Possui botões integrados (▲ e ▼) que incrementam ou decrementam o valor em 1 unidade.
  • Suporte a Teclado: Quando focado, as setas ArrowUp e ArrowDown do teclado executam a mesma função de incremento/decremento.

Eventos Emitidos

O fluxo de eventos é otimizado para garantir que apenas dados numéricos limpos sejam transmitidos.

  • algol-input: Disparado em tempo real durante a digitação, mas apenas se o caractere inserido respeitar a validação numérica. 
    document.querySelector('algol-input-number').addEventListener('algol-input', e => {
  console.log("Número sendo digitado:", e.detail.value);
});
  • algol-change: Disparado quando o campo perde o foco, o usuário aperta Enter, ou usa os botões/setas de incremento. É neste evento que a correção de min/max é garantida. 
    document.querySelector('algol-input-number').addEventListener('algol-change', e => {
  const { oldValue, value } = e.detail;
  console.log(`Valor matemático consolidado: [${value}]`);
});
