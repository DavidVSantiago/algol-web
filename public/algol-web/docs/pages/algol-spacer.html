<algol-spacer>

Visão Geral

Componente de layout invisível utilizado para criar espaçamentos verticais consistentes entre os elementos da página. Ele abstrai a necessidade de escrever CSS (margins/paddings) e aplica a altura matematicamente alinhada à escala global da biblioteca.

<!-- Cria um espaço de 2vw (multiplicado pelo scale-factor) -->
<algol-spacer value="2"></algol-spacer>

<!-- Espaçador responsivo (2vw no desktop, 1vw no mobile) -->
<algol-spacer value="2" valuebreak="1"></algol-spacer>

Atributos de Configuração

Atributo Descrição Valores Aceitos
value Define a altura do espaçamento para telas normais (desktop/tablet). Número (ex: 1.5, 2)
valuebreak Define a altura exclusiva para telas menores que o breakpoint mobile. Número (ex: 0.5, 1)

Motor de Cálculo Responsivo

Cálculo Automático: Você não precisa digitar a unidade de medida. Ao fornecer um número (ex: value="2"), o componente processa a conversão internamente e injeta o seguinte cálculo CSS:
height: calc(2vw * var(--scale-factor));

Eventos Emitidos

O algol-spacer herda todos os eventos do BaseComponent (como algol-click, algol-mouseenter). No entanto, por ser um elemento invisível focado puramente em afastar o conteúdo, a captura de eventos de interação raramente é necessária neste componente.
