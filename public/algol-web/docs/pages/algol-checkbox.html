<algol-checkbox>

Visão Geral

Componente de caixa de seleção (checkbox) customizado. Ele herda toda a lógica de estado, validação e formulário da classe BaseCheckable, entregando uma interface limpa, animada e perfeitamente alinhada ao tema global da biblioteca.

<algol-checkbox name="termos" required checked>Li e aceito os termos</algol-checkbox>

Atributos de Configuração

Graças à herança, o algol-checkbox possui suporte total aos seguintes atributos (detalhados na documentação do BaseCheckable):

  • checked: Define o estado inicial da caixa.
  • name: Chave de envio no formulário.
  • value: O valor enviado quando marcado (padrão "on").
  • disabled: Bloqueia a interação.
  • required: Impede o envio do formulário se desmarcado.
  • label: Texto embutido via atributo.

Flexibilidade de Texto (Label vs Slot)

Uso de Slot (Recomendado): Embora você possa usar o atributo label="Texto", o checkbox foi projetado para suportar injeção de HTML no seu interior via <slot>. Isso permite criar textos ricos contendo links, negritos ou ícones clicáveis diretamente ao lado da caixa de seleção.

Exemplo: <algol-checkbox>Aceito a <a href="#">Política de Privacidade</a></algol-checkbox>

Eventos Emitidos

O componente responde a cliques do mouse e barra de espaço (quando focado via Tab). O estado deve ser monitorado capturando o evento padronizado de troca:

  • algol-input: Disparado a cada inversão de estado. Acessando origin.checked você descobre se o usuário marcou ou desmarcou a caixa. 
    document.querySelector('algol-checkbox').addEventListener('algol-input', e => {
  const { origin, value } = e.detail;
  if (origin.checked) {
      console.log(`Marcado! Valor a ser enviado: ${value}`);
  } else {
      console.log("Desmarcado.");
  }
});
