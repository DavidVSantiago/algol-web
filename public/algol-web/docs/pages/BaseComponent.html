BaseComponent

Visão Geral

O BaseComponent é a classe abstrata fundamental da Algol UI. Ele não é instanciado diretamente no HTML, mas sim herdado por todos os outros componentes (como <algol-button>, <algol-input>, etc.).

A principal função desta classe é gerenciar o ciclo de vida, gerar getters e setters automáticos, gerenciar a transição de variáveis CSS e, mais importante, padronizar a emissão de eventos.

O Comportamento disabled

Regra Global: Se qualquer componente herdeiro possuir o atributo disabled, o BaseComponent intercepta nativamente todas as interações de clique e foco, acionando stopImmediatePropagation(). Nenhum evento customizado será emitido caso o componente esteja desativado.

Eventos Padrão (Custom Events)

Para garantir que os eventos atravessem o Shadow DOM com facilidade e entreguem dados estruturados e confiáveis, a biblioteca suprime os eventos nativos de mouse e foco, emitindo eventos customizados com o prefixo algol-.

Todos os eventos possuem bubbles: true e composed: true, e injetam seus dados na propriedade event.detail.

  • algol-click: Disparado ao clicar sobre o componente. 
    componente.addEventListener('algol-click', e => {
  const { origin, mouse: { x, y, offsetX, offsetY } } = e.detail;
  console.log("Clicado em:", origin, "Posição tela:", x, y, "Posição interna:", offsetX, offsetY);
});
  • algol-mousemove: Disparado durante a movimentação do cursor sobre o componente. 
    componente.addEventListener('algol-mousemove', e => {
  const { origin, mouse: { offsetX, offsetY } } = e.detail;
  console.log("Cursor movendo em:", origin, "Coords relativas:", offsetX, offsetY);
});
  • algol-mouseenter: Disparado quando o ponteiro do mouse entra na área do componente. 
    componente.addEventListener('algol-mouseenter', e => console.log("Mouse entrou:", e.detail.origin));
  • algol-mouseleave: Disparado quando o ponteiro do mouse sai da área do componente. 
    componente.addEventListener('algol-mouseleave', e => console.log("Mouse saiu:", e.detail.origin));
  • algol-focus: Disparado quando o componente recebe o foco (navegação por teclado ou clique). 
    componente.addEventListener('algol-focus', e => console.log("Foco ativado em:", e.detail.origin));
  • algol-blur: Disparado quando o componente perde o foco. 
    componente.addEventListener('algol-blur', e => console.log("Foco perdido em:", e.detail.origin));
