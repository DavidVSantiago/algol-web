<algol-radio>

Visão Geral

Componente de botão de opção (Radio Button) customizado. Utilizado quando o usuário precisa selecionar apenas uma opção dentre um conjunto mutuamente exclusivo. Assim como o checkbox, ele herda sua inteligência da classe BaseCheckable, mas com uma interface circular padrão para este tipo de controle.

<algol-radio name="plano" value="basico" checked>Plano Básico</algol-radio>
<algol-radio name="plano" value="premium">Plano Premium</algol-radio>

Agrupamento e Exclusividade

A Importância do Atributo name: Para que os botões de rádio funcionem corretamente (onde marcar um desmarca o outro automaticamente), todos os itens do mesmo grupo devem ter o exato mesmo valor no atributo name. A Algol UI rastreia o DOM ativamente para desmarcar os "irmãos" sempre que um novo item é clicado.

Atributos de Configuração

O algol-radio compartilha os mesmos atributos de seus componentes irmãos herdados do BaseCheckable:

  • name: Obrigatório para criar grupos de múltipla escolha onde apenas um vence.
  • value: O valor único que representa esta opção caso seja a escolhida e o formulário seja enviado.
  • checked: Define qual opção inicia pré-selecionada.
  • disabled: Bloqueia a interação do usuário com esta opção específica.
  • required: Exige que ao menos uma opção do grupo seja escolhida antes de submeter o formulário.

Flexibilidade de Texto (Slot)

Assim como no checkbox, o conteúdo de texto ao lado do rádio pode ser passado diretamente no "miolo" da tag (via <slot>), permitindo negritos, ícones ou links, em vez de usar apenas o atributo simples de label.

Eventos Emitidos

Sempre que o usuário seleciona uma opção não marcada, o evento de input é disparado:

  • algol-input: Disparado quando este rádio específico é ativado. Note que, pelo comportamento nativo do Radio Button, você geralmente não captura o evento de "desmarcar", apenas o de "marcar" a nova opção escolhida. 
    document.querySelectorAll('algol-radio[name="plano"]').forEach(radio => {
  radio.addEventListener('algol-input', e => {
      const { value } = e.detail;
      console.log(`Nova opção selecionada: ${value}`);
  });
});
