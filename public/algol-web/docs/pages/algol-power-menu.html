<algol-power-menu>

Visão Geral

Componente de barra de navegação (Header) completo e responsivo. Ele gerencia automaticamente o logotipo, a distribuição dos links e converte-se nativamente em um menu "hambúrguer" com overlay em telas de dispositivos móveis.

<algol-power-menu logo="logo.png" logolink="/" logosize="4vw">
    <li><a href="/home">Início</a></li>
    <li><a href="/sobre">Sobre</a></li>
    <li><div>Opção Customizada</div></li>
</algol-power-menu>

Diferencial de Renderização (Sem Shadow DOM)

useShadow = false: Ao contrário da maioria dos componentes da Algol UI, o Power Menu desativa o Shadow DOM. Isso é intencional. Ao renderizar no Light DOM, as tags <a> de navegação inseridas no menu podem herdar facilmente regras de estilização globais e scripts de roteamento (routers) de frameworks externos sem sofrer bloqueios de escopo.

Atributos de Configuração (Logotipo)

Atributo Descrição Valores Aceitos
logo URL ou caminho da imagem do logotipo. Texto (URL)
logoalt Texto alternativo de acessibilidade para o logotipo. Texto
logolink URL de destino ao clicar no logotipo (normalmente a home "/"). Texto (URL)
logowidth Define a largura nativa da tag img do logotipo. Número/Texto
logoheight Define a altura nativa da tag img do logotipo. Número/Texto

Atributos de Estilização

Atributo Variável Injetada Efeito Visual
logosize --power-menu-logosize Força a largura CSS do logotipo em telas Desktop/Tablet.
logosizebreak --power-menu-logosizebreak Força a largura CSS do logotipo exclusivamente em telas Mobile.

Montagem Automática da Grade (Links)

Sequestro de Filhos: Você não precisa se preocupar com o layout CSS dos botões. Escreva apenas tags <li> soltas dentro do componente. O Power Menu captura essas tags, aplica classes de interação (hover/active), conta quantos itens existem e calcula automaticamente o grid-template-columns para distribuir os links perfeitamente na barra superior.

Responsividade Integrada

O componente reage à variável global MOBILE_BREAKPOINT (padrão 600px). Ao atingir essa resolução:

  • A grade horizontal de botões é ocultada.
  • Um ícone de menu (hambúrguer) aparece no lugar.
  • Ao clicar no menu, os itens deslizam de cima para baixo acompanhados de um fundo escuro (overlay) que cobre a tela, impedindo cliques no restante da página.
  • Clicar no overlay fecha o menu suavemente.
