<algol-flex-item>

Visão Geral

O algol-flex-item atua como um contêiner flexível padrão. Ao ser instanciado, ele assume nativamente display: flex, com quebra de linha ativada (flex-wrap: wrap) e gap: 0. É a ferramenta principal para agrupar elementos horizontalmente ou atuar como uma célula dentro de um layout de grade maior.

<algol-flex-item distrib="between" padding="2">
    <div>Botão Voltar</div>
    <div>Botão Avançar</div>
</algol-flex-item>

Atributos de Layout Flexbox

Comportamento de Largura: Quando o atributo distrib é ativado, o componente automaticamente força seu tamanho para width: 100%, garantindo que a distribuição dos elementos ocupe todo o espaço horizontal disponível no elemento pai.
Atributo Descrição / Regra Injetada
distrib Controla a distribuição dos filhos (justify-content). Valores aceitos: start, center, end, between, around, evenly.
padding Espaçamento interno. Converte números soltos para vw automaticamente. (Ex: "1 2" vira 1vw 2vw).

Integração Híbrida (Uso dentro de Grids)

Se este componente for inserido como filho direto de um <algol-grid-layout>, ele suporta atributos de expansão de célula:

Atributo Regra CSS Injetada
colspan grid-column-end: span [valor]
rowspan grid-row-end: span [valor]

Controles Responsivos

Os atributos abaixo ativam Media Queries que sobrescrevem os valores nativos exclusivamente quando a tela atinge a largura mobile estipulada pela engine:

  • colspanbreak: Altera a expansão de colunas em telas pequenas.
  • rowspanbreak: Altera a expansão de linhas em telas pequenas.
  • paddingbreak: Redefine o espaçamento interno (padding) na versão mobile.

Sistema de Background (Imagem)

Auto-Expansão de Imagem: Ao fornecer o atributo img, o componente não apenas aplica a imagem de fundo, mas também injeta obrigatoriamente width: 100% e height: 100% para garantir a renderização adequada do fundo dentro do contêiner pai.
Atributo Função (Mapeamento CSS)
img URL da imagem de fundo (background-image: url(...)).
imgsize Tamanho do fundo (ex: cover, contain).
imgpos Posição da imagem (ex: center, top right).
imgrepeat Repetição (ex: no-repeat, repeat-x).
imgattach Fixação do fundo (ex: fixed, scroll).
