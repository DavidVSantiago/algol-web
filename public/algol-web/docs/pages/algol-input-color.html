<algol-input-color>

Visão Geral

Componente para seleção de cores. Ele oculta a interface nativa (e visualmente inconsistente) de escolha de cor dos navegadores, substituindo-a por um painel customizado elegante da Algol UI, mas ainda acionando o seletor de cores robusto do sistema operacional do usuário.

<algol-input-color label="Cor Principal" value="#ff0000"></algol-input-color>

Atributos de Configuração

Por ser focado em uma única tarefa visual, este componente possui um conjunto reduzido de configurações:

Atributo Descrição Valores Aceitos
label Texto descritivo exibido acima da caixa de cor. Texto
value A cor atual selecionada. Se omitido, o padrão inicial será vermelho (#f00). Texto (Hexadecimal) (ex: #336699)
disabled Bloqueia a interação, aplica efeito escurecido e desabilita o foco via teclado. Booleano (presença do atributo)

Acessibilidade e Estrutura

Navegação por Teclado: Apesar da tag <input> nativa estar oculta (display: none), a caixa customizada possui índice de tabulação (tabindex="0"). O componente intercepta o foco e aguarda o uso das teclas Espaço ou Enter para abrir a paleta de cores automaticamente, garantindo 100% de acessibilidade.

Eventos Emitidos

O fluxo de eventos herdado do input padrão permite rastrear tanto a escolha em tempo real quanto a escolha final:

  • algol-input: Disparado em tempo real enquanto o usuário arrasta o cursor pela paleta de cores do sistema operacional. Útil para "live previews". 
    document.querySelector('algol-input-color').addEventListener('algol-input', e => {
  console.log("Visualizando cor em tempo real:", e.detail.value);
});
  • algol-change: Disparado quando o usuário clica fora ou fecha a janela do seletor de cores, confirmando a seleção. 
    document.querySelector('algol-input-color').addEventListener('algol-change', e => {
  const { oldValue, value } = e.detail;
  console.log(`Cor confirmada! Mudou de [${oldValue}] para [${value}]`);
});
